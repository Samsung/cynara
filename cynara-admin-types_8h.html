Wildcard

Can replace client, user or privilege name. WILDCARD matches any string during check procedure from libcynara-client.

#define CYNARA_ADMIN_WILDCARD   "*"
 
Name of Default Bucket

Default bucket - the one that check starts in. Default bucket cannot be removed, although its default policy (which originally is set to DENY) can be changed.

#define CYNARA_ADMIN_DEFAULT_BUCKET   ""
 
Any

Can replace client, user or privilege name. ANY matches any string (including WILDCARD) during:

  • policy removal with cynara_admin_erase() function
  • listing policies from a single bucket.

Using ANY as default policy for bucket or as policy type of inserted policy record is forbidden and will cause CYNARA_API_INVALID_PARAM error.

#define CYNARA_ADMIN_ANY   "#"
