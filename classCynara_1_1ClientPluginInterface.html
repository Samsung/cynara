Detailed Description

A class defining external plugins interface.

These plugins work inside of cynara client library. They interpret PolicyResult returned by cynara in terms of:

  • cacheability - tells, whether value should be cached (for e.g. policyType like ALLOW_ONCE should not be cached)
  • usability - whether cache entry can still be used (for e.g. policy allowing access for given type)
  • value - translates PolicyResult to CYNARA_API_ACCESS_ALLOWED or CYNARA_API_ACCESS_DENIED

Plugin implementing ClientPluginInterface must implement ExternalPluginInterface. Creation/destruction functions with signatures compatible to Cynara::create_t and Cynara::destroy_t must be provided as factories of ClientPluginInterface.
